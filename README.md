# نظام إدارة الحجوزات والمناسبات — أم فارس

تطبيق ويب كامل (Full-stack) لإدارة حجوزات المناسبات والضيافة (صبابات، عاملات، سائق)، واجهة عربية RTL.

## التقنيات
- **الواجهة الخلفية:** Node.js + Express + SQLite (`node:sqlite` المدمج) + JWT
- **الواجهة الأمامية:** React + Vite + Tailwind CSS + Recharts

## المميزات
- تسجيل دخول آمن (JWT + bcrypt)
- تقويم رئيسي بعرض أسبوعي/شهري مع التنقل بين الفترات
- إضافة حجز جديد بكامل الحقول (مناسبة، معاميل، صبابات، عاملات، مشروبات إضافية...)
- بطاقات اليوم مع مؤشرات (صبابات / عاملات / سائق) بالأخضر والأحمر
- تفاصيل الحجز الكاملة: تعديل، حذف، إلغاء الموعد، إضافة خصم، اختيار الموظفين المتاحين
- تأكيد إنهاء الحجز (اكتمال الدفع + توزيع الإكراميات)
- إدارة الحجوزات (مع حد أقصى للحجوزات اليومية)
- إدارة الخدمات والباقات
- إدارة العملاء وسجل طلباتهم
- إدارة المالية: رسوم بيانية لصافي الربح والمدخول + فلاتر
- إدارة الموظفين (تصنيفات متعددة + أجر)
- إدارة المستخدمين (تفعيل/إيقاف + تغيير كلمة السر)
- إشعارات داخل التطبيق (تذكير قبل ٣ أيام / يوم، تنبيه عدم حجز الموظفين، تأكيد إنهاء اليوم)

## التشغيل

```bash
# 1) تثبيت كل الحزم (الجذر + الخادم + الواجهة)
npm run install:all

# 2) تشغيل الخادم والواجهة معاً (وضع التطوير)
npm run dev
```

- الواجهة: http://localhost:5173
- الـ API: http://localhost:4000

## للنشر (Production)
```bash
npm run build      # يبني الواجهة في client/dist
npm start          # الخادم يقدّم الواجهة والـ API على المنفذ 4000
```
